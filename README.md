# OW2 克制助手

守望先锋2（Overwatch 2）英雄克制关系查询微信小程序，帮助玩家选择合适的英雄应对敌方阵容。

## 功能特性

- 敌方阵容选择（最多6人）
- 智能推荐克制英雄
- 详细的克制关系分析
- 技能关键词高亮显示
- 支持全部44个英雄

## 文件分类说明

### 配置文件（5个）
| 文件 | 用途 |
|------|------|
| app.js | 初始化应用，定义全局数据 |
| app.json | 页面路由、导航栏样式配置 |
| app.wxss | 通用样式类（flexbox、颜色、动画等） |
| project.config.json | 微信开发者工具配置、AppID |
| sitemap.json | 小程序爬虫开放配置 |

### 页面文件（8个）
- **index 主页面**：英雄选择界面，分为坦克/输出/辅助三个区域，支持选择敌方阵容并查看推荐英雄
- **detail 详情页**：展示单个英雄的克制分析详情，包括风险提示和克制理由

### 数据文件（1个）
- **utils/data.js**：包含所有核心数据
  - `HERO_DATA`：44个英雄的基本信息（坦克13、输出19、辅助12）
  - `COUNTER_LOGIC`：每个英雄的克制关系和详细说明
  - `KEYWORDS`：100+个技能关键词（用于高亮显示）
  - `getHeroImagePath()`：获取英雄图片路径的工具函数

### 资源文件（44个）
- **images/**：所有英雄的头像PNG图片

## 开发说明

### 环境要求
- 微信开发者工具

### 运行项目
1. 打开微信开发者工具
2. 导入项目目录
3. 编译运行

### 项目配置
- AppID 在 `project.config.json` 中配置
- 页面路由在 `app.json` 中配置

## 技术栈

- 微信小程序原生框架
- WXML + WXSS + JavaScript
- 数据驱动的克制关系计算
